\section{Geometrical Introduction}

Suppose $\Gamma$ is a connected graph where each node has even degree. Prove
that every graph obtained by removing one edge from $\Gamma$ is still connected.

\begin{proof}
    To show that $\Gamma$ is still connencted, we have to show that there exists 
    a walk from $v_0$ to $v_k$ where $v_0, v_k$ are the nodes between which an 
    edge has been removed. Since before the removal we have an Eulerian graph,
    we consider the edge that is removed to be the last edge in the walk.
    We still have a walk that reaches every point in the graph, since every edge
    has been traversed. 
\end{proof}
